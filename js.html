Aprendendo JavaScript do Zero

Guia básico para deixar o site interativo

1. O que é JavaScript?

JavaScript é a linguagem de programação usada para dar funcionalidade e interatividade aos sites. Com ele você pode criar botões que reagem a cliques, animações, formulários inteligentes e muito mais.

2. Como usar JavaScript

  • Interno: dentro da tag <script> no HTML.
  • Externo: em um arquivo separado .js.

3. Exemplo básico

Um alerta ao abrir a página:

<script>
  alert("Bem-vindo ao meu site!");
</script>

4. Manipulando conteúdo

HTML:

<p id="texto">Texto original</p>
<button onclick="mudarTexto()">Clique aqui</button>

JavaScript:

function mudarTexto() {
  document.getElementById("texto").innerText = "O texto mudou!";
}
