Aprendendo CSS do Zero

Guia básico para deixar seu site bonito

1. O que é CSS?

CSS significa Cascading Style Sheets. Ele é usado para dar estilo ao HTML, como cores, tamanhos, fontes e layouts.

2. Como usar CSS

Existem três formas principais de usar CSS:

  • Inline: direto dentro da tag HTML.
  • Interno: dentro da tag <style> no HTML.
  • Externo: em um arquivo separado .css.

3. Exemplos

Exemplo Inline:

<p style="color: blue;">Texto azul</p>

Exemplo Interno:

<style>
  p { color: red; }
</style>
<p>Texto vermelho</p>

Exemplo Externo:

/* style.css */
body {
  background-color: lightgray;
  font-family: Arial;
}

4. Propriedades mais usadas

  • color: cor do texto
  • background-color: cor de fundo
  • font-size: tamanho da fonte
  • margin: espaço externo
  • padding: espaço interno
  • border: borda do elemento

5. Exemplo prático

HTML:

<h1>Olá, mundo!</h1>

CSS:

h1 {
  color: blue;
  text-align: center;
}

Resultado: Um título azul e centralizado.

6. Conclusão

O CSS transforma um site simples em algo visualmente agradável. Ele trabalha junto com o HTML para criar páginas profissionais.
