Aprendendo HTML do Zero

Guia básico para quem nunca programou

1. O que é HTML?

HTML significa HyperText Markup Language. Ele é usado para criar a estrutura de páginas da web, como textos, imagens, links e listas.

2. Estrutura básica de uma página HTML

<!DOCTYPE html>
<html>
  <head>
    <title>Título da Página</title>
  </head>
  <body>
    Aqui vai o conteúdo da página!
  </body>
</html>

3. Principais tags

  • <h1> até <h6>: Títulos (h1 é o maior, h6 o menor)
  • <p>: Parágrafos de texto
  • <a>: Links
  • <img>: Imagens
  • <ul> <li>: Listas

4. Exemplos práticos

Título e parágrafo:

<h1>Bem-vindo ao meu site</h1>
<p>Esse é um parágrafo de exemplo.</p>

Link:

<a href="https://www.google.com">Ir para o Google</a>

Imagem:

<img src="https://via.placeholder.com/150" alt="Exemplo de imagem">

Lista:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

5. Conclusão

O HTML é a base de todo site. Com ele você cria a estrutura. Depois, pode usar CSS para estilos e JavaScript para interatividade.
